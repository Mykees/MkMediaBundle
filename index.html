The goal of this bundle is to allow you to associate media with several entities

Functionality :

  • Easy to use and configure

    • This MediaBundle is very easy to initialize and configure. Add your media and retrieves them very simply without so much code.

  • Drag and Drop System

    • This MediaBundle use a drag and drop system by Dropzone.js. It's very easy to add one or multiples media for your content

  • WYSIWYG or Iframe

    • To add media on your content, you can add an iframe uploader or if you have write articles by example, you can add a wysiwyg(Tinymce) wich integrates a media uploader.

  • Resize your media

    • This MediaBundle integrates a Resize media system. You can configure very easily different size for your images. The bundle use Imagine library.
